About Edit Distance

The edit distance (Wikipedia) between two strings is the mininum number of changes required to transform one string into another. This implementation calculates the Levenshtein distance (Wikipedia) which allows insertion, substitution, and deletion. Each operation has a cost associated with it:

  • Insertion: 1
  • Substitution: 0 if the characters are the same, 2 otherwise
  • Deletion: 1
